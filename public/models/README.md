# Modèles 3D

- **Soldier.glb** — personnage humanoïde riggé avec animations Idle / Walk / Run.
  Source : exemples officiels Three.js (modèle de Tomás Laulhé, modifié par Don McCurdy,
  licence CC0). Utilisé comme personnage joueur réaliste.

Pour remplacer le personnage par le tien : dépose un `.glb` riggé ici (ex. avatar
Ready Player Me ou export Mixamo converti en glb) et change le chemin dans
`src/game/PlayerController.js` (`/models/Soldier.glb`).
